# Description
Building a telepresence robot, leveraging WebRTC, ROS, and Nodejs. Built on top of a iRobot Create 3.

# Install
- [Pi hookup and ROS install to create 3](https://iroboteducation.github.io/create3_docs/setup/pi4humble/)
